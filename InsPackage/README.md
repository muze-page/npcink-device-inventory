# 介绍

## 用途

局域网内，收集当前设备信息上传至指定设备上，进行统一管理和展示
局域网外，收集当前设备信息上传至指定网络上，进行统一管理和展示

## 大概思路

- 每台设备具有唯一标识符，可备注使用人员。
- 统计所有硬件信息
- 资产管理
- 硬件大盘
